Dear FortranGIS users,
	a new version of FortranGIS has been released, it includes a new and 
almost complete interface to GDAL library, allowing to read and write 
all the raster formats supported by GDAL. GDAL interface still requires 
testing, so please report any problem. Following the link to the online 
documentation, you can find some test programs which can be used as an 
example of application, both for Shapelib and for GDAL:

http://fortrangis.berlios.de/doc/index.html

The Shapelib API has not changed in this update.

	best wishes, Davide Cesari
