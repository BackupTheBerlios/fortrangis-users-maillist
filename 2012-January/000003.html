Hi Marcos, thank you for opening the mailing list; yes the project is 
alive, although not very active, I admit, I make some commits from time 
to time.

The shapelib interface is quite stable and tested, you can use the 
shapelib_test.f90 source as an example of use; the gdal part is less 
mature, I hope to commit soon a new and more complete version with 
automatically-generated interfaces, which will be more complete.

I am interested in hearing the needs of users, so please tell me how you 
would plan to use FortranGIS, which operations you need most, which kind 
of files etc.

Notice that, at the moment, FortranGIS builds only on Unix-like systems.

Best regards, Davide
