Hi Marcos,

some weeks ago, I had a similar problem with the same Ubuntu version and 
libfortrangis. My problem was that the compiled program was not able to find 
the library anymore and the linker could not find lgdal. It worked, when I 
compiled libfortrangis giving the option --disable-shared to the ./configure 
command.
In order to fix the linker problem, the compilation command was then
gfortran -I/usr/local/include/ `gdal-config --cflags` gdal_test.F90 `gdal-
config --libs` -lfortrangis -lfortranc

I hope this helps you to fix your problem

Best wishes

Lukas
