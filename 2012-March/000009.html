> Yes, I confirm, the default installation path for libraries in the build system, /usr/local/lib, is not usually in the default search path for shared libraries in Linux, so either you compile only static libraries, configuring with --disable-shared, as Lukas said, or you set at runtime:
> 
> export LD_LIBRARY_PATH=/usr/local/lib
> or
> LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/local/lib
> if it is already set
> 
> in order to dynamically link the shared libraries you installed. The same holds if you installed on a non-standard location, such as your own home directory.
