Hello Dominik,
	The package is based on autotools, so it does not build on Windows as 
it is, maybe under Cygwin, but it is not the same thing. But in 
principle most of FortranGIS is actually a "thin standard Fortran 2003 
layer" over other C/C++ libraries available under Windows, so, at least 
as a first try, it may be a matter of compiling by hand some Fortran 
sources with a F2003 compiler, but it is possible that some 
modifications are required. If you tell me which compilers you are using 
I can make a more accurate guess.
	Davide
